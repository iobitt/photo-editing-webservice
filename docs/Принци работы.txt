@startuml
Клиент -> Контроллер: Запрос на обработку изображения
Контроллер -> База_данных: Создаёт запись в таблице jobs
База_данных -> Контроллер: Возвращает job_id
Контроллер -> Очередь_фоновых_задач: Создаёт фоновую задачу
Контроллер -> Клиент: Возвращает идентификатор созданной задачи job_id

Воркер --> Сервис: Вызывает и передаёт параметры, полученные контроллером
Сервис --> Воркер: Возвращает результат в виде изображения или текстового результата
Воркер --> База_данных: Записывает результат операции

Клиент -> Контроллер: Запрашивает статус и результат операции по job_id
Контроллер -> База_данных: Передаёт job_id
База_данных -> Контроллер: Возвращает запись по job_id
Контроллер -> Клиент: Возвращает статус операции и результат операции, если он готов
@enduml
